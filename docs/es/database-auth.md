# Base de Batos Auth
La base de datos auth cuenta actualmente con 14 tablas, que almacenan principalmente información relacionada con las cuentas de usuario. Aunque algunas de ellas tienen otras funcionalidades. La última sincronización se realizó: [01-10-2021](https://github.com/azerothcore/wiki/issues/338). Si detectas que alguna tabla está desactualizada, puedes crear un [issue](https://github.com/azerothcore/wiki/issues) dentro de nuestro repositorio o hacer un [pull request](https://github.com/azerothcore/wiki/pulls) con la modificación de la información.
